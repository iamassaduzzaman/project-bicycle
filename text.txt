This Project is called Bicycle Management for Our Green Campus 
using Data Structure

Some key features of this project: 
1. Create new account for new user
2. User Profile
3. Total Cycle on Field
4. Cycle Availibilty
5. Rent a Cycle
6. Return a Cycle
7. User Log In
8. User Log Out
